Author: Luc-Tanton Tran

Purpose: New dataset variable descriptions.

Description of variables:


Category 1: Distances regrouped categorically using quartiles (close, median close, median far, far)

- distwater_cat = ...from nearest water source
- disturb_cat	= ...from nearest urban area
- distfarm_cat	= ...from nearest livestock farm
- distpoul_cat	= ...from nearest poultry farm

Category 2: Presence or Absence of...

- Subcategory 1: ...an allele
  5239s1_A
	5239s1_T
	7259s1_A
	7259s1_T
	TLR4_1_A
	TLR4_1_G
	TLR4_2_A
	TLR4_2_G
	TLR4_3_C
	TLR4_3_T
	TLR4_4_A
	TLR4_4_C
- Subcategory 2: ...a haplotype
  TLR4_Prot_1
	TLR4_Prot_2
	TLR4_Prot_3
	TLR4_Prot_4
	
Category 3: Number of copies of a SNP allele, eg. 5239s1.T = 2 is a homozygote TT:
	5239s1.T
	7259s1.A
	TLR4_1.G
	TLR4_2.A
	TLR4_3.C
	TLR4_4.C
